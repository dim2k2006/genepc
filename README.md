# genepc

> Generates EPC from EAN

## Install

```sh
npm i @dim2k/genepc-cli -g
```

## Usage

```sh
# EAN: 52578029

genepc 52578029

# Result: EPC 3000000140E9280000000000 is in your clipboard.
```
